# ⚡ Tezkor Boshlash

## 1️⃣ O'rnatish (1 daqiqa)

```bash
cd /home/azam/Desktop/yaratish/play_deploy
./start.sh
```

## 2️⃣ Ishga Tushirish (30 soniya)

### Terminal 1 - Backend
```bash
./run-backend.sh
```

Kutish: Backend http://localhost:8000 da ishga tushdi

### Terminal 2 - Frontend  
```bash
./run-frontend.sh
```

Kutish: Frontend http://localhost:5173 da ishga tushdi

## 3️⃣ Brauzerda Ochish

```
http://localhost:5173
```

## 4️⃣ Birinchi Ishlatish (5 daqiqa)

### A. Ro'yxatdan o'tish
- Email va parol kiriting
- "Ro'yxatdan o'tish" tugmasini bosing

### B. Service Account Ulash
- Account nomi kiriting
- JSON kalit faylini yuklang
- "Ulash" tugmasini bosing

### C. Ilovalarni Sinxronlash
- "Ilovalarni Sinxronlash" tugmasini bosing
- Ilovalar ro'yxati ko'rinadi

### D. Store Listing Tahrirlash
- Ilovani tanlang
- Matnlarni kiriting (nom, qisqa va to'liq ta'rif)
- "Barcha Tillarga Tarjima" bosing
- Rasmlar yuklang
- "Play Console'ga Yuborish" bosing
